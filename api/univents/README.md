# 🌍 Univents Platform Overview

Univents is a full-cycle academic event management and engagement platform supporting organizers, staff, speakers, attendees, sponsors, and institutions across recurring event series.

---

# 👩‍💼 For Organizers

## 🎟️ Event & Ticketing

### Event Structure

* Only approved users can create events
* Events can be structured as **series with multiple editions**

    * Series = long-term identity (brand, history)
    * Edition = specific year/instance (schedule, tickets, finances, attendance)

### Products & Sales

* Sell:

    * Participation tickets
    * Tiered ticket levels
    * Merchandise
    * Paid activities

### Ticket Capabilities

Tickets can:

* Grant access levels
* Include **tokens**
* Unlock specific activities

---

## 🗓️ Activity & Program Management

### Activity Creation

Supports:

* Talks
* Workshops
* Panels
* Minicourses
* Labs
* Field trips
* Paid or free sessions

Each activity can define:

* Participant limits
* Payment type (free, paid, token-based)

### Activity Format Intelligence (Analytics)

* Tag activities by format (talk, workshop, etc.)
* Compare formats across editions:

    * Attendance
    * Ratings
    * Engagement levels

### Ways Participants Join

* Direct registration
* Interest marking
* Payment
* Token spending

### Token Economy

* Tickets may include tokens
* Tokens can be used instead of money for premium sessions
